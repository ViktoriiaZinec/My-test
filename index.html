ЗАВДАННЯ 1

Як відправити код в пустий непов'язаний репозиторій git@example.com:example/test.GIT

  1. В терміналі знаходжу свій каталог і прописую команду git init, яка створює каталог .git і наповнює його файлами, що формують структуру git репозиторію (не видно в vs code, з терміналу ввести ls -a (--all))
  2. Комічу свої файли: git add . , git commit
  3. Додаю посилання на віддалений репозиторій git remote add origin git@example.com:example/test.GIT (SSH репозиторія)
  4. Відправляю свій коміт до віддаленого репозиторію -git push origin HEAD:master
