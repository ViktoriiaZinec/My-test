Напишите метод arrayToObject, который превращает массив в объект (использовать рекурсию).

Пример:

var arr = [['name', 'developer'], ['age', 5], ['skills', [['html',4], ['css', 5], ['js',5]]]]; arrayToObject(arr)

// Outputs: { name: 'developer', age: 5, skills: { html: 4, css: 5, js: 5}}
